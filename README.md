# Performance Evaluation Matrix using Python

## Project Overview

This project evaluates intern performance using a KPI-based system. It automates the calculation of performance metrics, generates monthly reports, and visualizes intern performance through charts.

## Objective

- Evaluate intern performance using Key Performance Indicators (KPIs)
- Automate KPI calculations with Python
- Generate monthly reports
- Visualize performance using charts

## KPIs Used

- Task Completion Time (30%)
- Project Quality (40%)
- Mentor Feedback (30%)

## Technologies Used

- Python
- Pandas
- Matplotlib
- OpenPyXL
- Jupyter Notebook

## Features

- KPI Calculation
- Performance Rating
- Monthly Report Generation
- Automatic Chart Creation
- CSV & Excel Export

## Performance Categories

| KPI Score | Rating |
|-----------|---------|
| 90-100 | Excellent |
| 80-89 | Very Good |
| 70-79 | Good |
| Below 70 | Needs Improvement |

## Outputs

- CSV Report
- Excel Report
- Performance Charts
